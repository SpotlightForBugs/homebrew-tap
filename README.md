<h1 align="center">SpotlightForBugs Homebrew Tap</h1>

<p align="center">
  Homebrew formulae for <a href="https://github.com/SpotlightForBugs">SpotlightForBugs</a> projects on macOS and Linux.
</p>

```bash
brew tap SpotlightForBugs/tap
```

Formulae in `Formula/` are updated automatically when upstream projects release.

## Formulae

### `kramli`

CLI client for [Kramli](https://kramli.de) — shopping lists, todos, and shared lists from your terminal.

```bash
brew install SpotlightForBugs/tap/kramli
```

```bash
kramli login
kramli lists list
kramli items list <LIST_ID> --open
```

Source and other install methods: [kramli-cli](https://github.com/SpotlightForBugs/kramli-cli)
